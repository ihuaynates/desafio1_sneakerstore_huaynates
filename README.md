# Descripción del proyecto

Ecommerce de venta de sneakers en Perú.

## Scripts para Ejecutar

Ejecutar los siguiente comandos:

### `clone`

Se debe clonar el repositorio.

### `npm install`

Instalar dependencias.

### `npm start`

Ejecutar el servidor de desarrollo.
